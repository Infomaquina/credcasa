
## Considerações:
- Backend com função generica (controller+model) em cada classe foi uma solução paliativa...
- O ideal seria chamar apenas os métodos solicitados pelo ajax através de um sistema de rotas já no startup. Porém, essa investida mudaria um pouco a estrutura já existente do projeto. Acredito que perderia o foco por não fazer parte do objetivo.
- Infelizmente precisaria de mais tempo para implementar os recursos de busca, filtro, ordenação e paginação da tabela de leads.
